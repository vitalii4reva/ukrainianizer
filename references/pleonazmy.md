# Плеоназми

Зайві слова, що дублюють значення. AI особливо схильний до них через прагнення «уточнити».

## Класичні плеоназми

| ❌ Плеоназм | ✅ Правильно |
|-------------|-------------|
| вільна вакансія | вакансія |
| пам'ятний сувенір | сувенір |
| власна автобіографія | автобіографія |
| спільна співпраця | співпраця |
| передовий авангард | авангард |
| повністю завершений | завершений |
| головний пріоритет | пріоритет |
| вперше дебютувати | дебютувати |

## Приклад

**До:**
> Головний пріоритет — повністю завершити спільну співпрацю, щоб вперше дебютувати з кінцевим результатом.

**Після:**
> Пріоритет — завершити співпрацю, щоб дебютувати з результатом.
